💳 ATM Interface System (Java)
📌 Description

This is a console-based Java application that simulates basic ATM functionalities. Users can log in using a User ID and PIN and perform operations like deposit, withdrawal, transfer, and view transaction history.

🚀 Features:
🔐 User authentication (User ID & PIN)
💰 Deposit money
💸 Withdraw money
🔁 Transfer funds
📜 Transaction history
❌ Exit option

🛠️ Technologies Used:
Java (Core Java)
Object-Oriented Programming (OOP)
VS Code / Any Java IDE

▶️ How to Run in VS Code:
Install Java JDK
Install VS Code
Install Java Extension Pack in VS Code

Copy the code into a file named:

ATMInterface.java

Open terminal in VS Code and run:

javac ATMInterface.java
java ATMInterface
